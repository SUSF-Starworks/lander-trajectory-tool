# lander-trajectory-tool

Built for SUSF Starworks as a preliminary design tool.
Its aim is to inform key parameters such as tank sizing and throttle profiles.
Built on my own vtvl control sim https://github.com/dfps16/vtvl-sim

# Usage
Clone the repository on your machine.
1. Install uv: https://docs.astral.sh/uv/getting-started/installation/
2. Run [uv sync] at repo root, installs all necessary packages and created a Python venv. 
3. Run app.py, through terminal or in your IDE of choice
4. Usage manual will come at some point
